# EventGenie AI 🎉
EventGenie AI is an AI-powered event assistant web application built using Streamlit.

## Features
- 🎯 Suggested Events recommendations
- 🗓 Today's Schedule timeline
- 🔥 Don't Miss highlights
- 🤖 AI Assistant for event-related queries
- 📍 Event Details section
- ✨ Interactive UI with glassmorphism design and dynamic background

## Tech Stack
- Python
- Streamlit
- HTML/CSS (custom styling)
- (Optional) OpenAI API integration

## Project Structure
```text
app.py
background.jpg
requirements.txt
```

## How to Run
```bash
streamlit run app.py
```

## Use Case
EventGenie AI helps users:
- discover relevant events
- check schedules
- find important sessions
- interact with an event assistant
